# オタ活支援アプリOtakker
オタク活動を日記のように残せるアプリケーションです。

## 機能一覧
ユーザー：
ログイン、ログアウト、サインアップ
投稿：
一覧、作成、削除

## 使用技術
Django  
Postgres  
Docker  
Burma
